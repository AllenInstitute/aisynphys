aisynphys documentation
=======================

The code and tools described here are used to access data generated by the `Synaptic Physiology project <https://portal.brain-map.org/explore/connectivity/synaptic-physiology>`_ at the `Allen Institute for Brain Science <https://alleninstitute.org/what-we-do/brain-science/>`_.

This documentation can be `viewed online at readthedocs.org <https://aisynphys.readthedocs.org>`_.

The `main code repository <https://github.com/alleninstitute/aisynphys>`_ and `documentation source <https://github.com/AllenInstitute/aisynphys/tree/documentation>`_ are hosted on GitHub.
